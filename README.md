## Features
- User authentication prototype with Login, Signup, and Forgot Password screens.
- First-time user onboarding with major selection and “Continue as Undecided” option.
- Tutorial flow explaining the main navigation icons and app sections.
- Dashboard showing degree progress, upcoming deadline, and recommended next step.
- Degree Pathway page showing major, semester, completed courses, current courses, remaining requirements, and major change flow.
- Semester Planning tool with semester selection, recommended courses, custom course entry, selected course summary, prerequisite validation simulation, and save confirmation modal.
- Deadlines page with upcoming deadlines, priority levels, detail modal, required checklist, consequences, resources, completion validation, and task completed feedback.
- Help/Resources page with helpful campus links and redirect simulation modal.
- Profile page with account information, academic information, password change action, sign out, and app preference checkboxes.
- Responsive styling, reusable CSS classes, modal feedback states, disabled button tooltip, and simulated local storage behavior.
- Reusable components including bottom navigation, semester selector, primary/secondary/small buttons, and back/cancel button.
